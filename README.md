# React Todo Application

This is a simple React Todo application developed as a frontend trial task.
The application allows users to create, view, edit, delete, and update the
status of todos along with image upload functionality.

---

## Features
- Add Todo with image upload
- View Todo list
- Edit Todo (including image update)
- Delete Todo with confirmation
- Mark Todo as pending or completed
- Data stored using browser localStorage
- No backend required

---

## Technologies Used
- React 18
- React Router DOM
- Vite
- JavaScript
- CSS
- Browser localStorage

---

## Project Setup Instructions

### Step 1: Install dependencies
'''bash'''
npm install
npm install react-router-dom

### Step 2: Run the development server
npm run dev

### Step 3: Open in browser
http://localhost:5173

 ---

### Project Approach

1. This project is built using React functional components and hooks.
2. React Router is used for navigation between pages.
3. Todos are stored in browser localStorage since no backend is used.
4. Images are stored as Base64 data URLs.
5. The application follows a clean folder structure with separation of components,
pages, and services.

#### Folder Structure

src/
components/
pages/
services/
main.jsx
